# Task overview

Create an MS Outlook VBA macro which looks at a private group I am member of and exports all the emails in it into a single text file. No attachments should be exported, only the from, to, cc and subject fields and the whole text content.

Put the result into a new file with VBA extension.

The exact name of the group is "ai-edih-archivum". The folder name is "Beérkezett üzenetek".

# Output structure

Use a separator and a header format in the text file as follows:

---[New exported AI-EDIH-ARCHIVUM email]---
From: xxxxxxxxx
To: xxxxxxxx; yyyyyyyy
Cc: xxxxxxxx; yyyyyyyy

Message body
